Hi! This is a web page to test all your audio input options.

First, hit Setup Recording and allow the microphone to be used by this website. After that, try all your input sources one by one and hit record. It will record for 8 seconds and then auto-play afterwards. You can listen to every recording again and compare. If you plug in more options, hit Refresh devices to make them available.
